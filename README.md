# Job Quick Guide (JQG)

FF14のスキルを素早く確認するためのWebツールです。

レベルに応じて使用可能なスキルを表示し、
軽減・ヒールスキルを中心に確認できるように作りました。

## Features
・ジョブ切り替え  
・レベルスライダーによるスキル表示  
・スキルカードUI  
・タグ付きスキルデータ
